# MySalon

A salon management app to manage hair services, products, customer details, and staff attendance efficiently.

## Features
- **Hair Services Management:** Add, view, and update all salon services.
- **Product Management:** Track salon products and inventory.
- **Customer Tracking:** Maintain customer details for better service.
- **Staff Attendance:** Record and manage staff attendance easily.
- **Simple Interface:** Easy-to-use web interface for quick access.

## Technologies Used
- HTML, CSS, JavaScript
- JSON for data storage (can be replaced with a database later)
